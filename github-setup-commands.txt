# COMANDOS PARA CONECTAR AO GITHUB
# Execute estes comandos na ordem, substituindo SEU-USUARIO pelo seu username do GitHub

# 1. Adicionar repositório remoto
git remote add origin https://github.com/SEU-USUARIO/aula-completa.git

# 2. Verificar se foi adicionado corretamente
git remote -v

# 3. Fazer push inicial
git push -u origin main

# EXEMPLO PRÁTICO:
# Se seu username for "joaosilva", o comando seria:
# git remote add origin https://github.com/joaosilva/aula-completa.git
